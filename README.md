# Guns of Icarus Log Tool
A tool that helps with saving logs for Guns of Icarus.

## Current Features
#### Output Log
- Manually save log.
- Autosave: 
  - Automatic: 
  
    Everytime the game is closed, save the log with timestamp.
  - Prompt: 
  
    Everytime the game is closed, open the Save Log dialog.
